# 晗晗的 Python 小课堂 💕

一个专为晗晗准备的 Python 入门学习网页：既有爱意，又容易坚持学下去。

## 方案说明

### 表达爱意

- **标题与文案**：页面标题为「晗晗的 Python 小课堂」，副标题为「写给你的代码情书」。
- **欢迎语**：首页有一段写给晗晗的欢迎和一句小情话。
- **每节鼓励语**：每个小节都有一句单独的鼓励/解释，用轻松的语气陪她学。
- **课程内容**：示例代码里会自然出现「晗晗」、小情诗等，学语法的同时也能感受到心意。
- **视觉风格**：粉白配色、圆角卡片、轻微动效，整体偏温馨可爱。

### 让她容易学下去

- **零环境**：用 Pyodide 在浏览器里运行 Python，不用装 Python、不用配环境，打开网页就能写、能运行。
- **短章节**：从「第一行 print」到「变量、字符串、列表、循环、函数」，再到一个「专属小情诗」小项目，每节只讲一个点，负担小。
- **即改即跑**：每节都有「试试运行」按钮，弹出框里可以改代码并运行，立刻看到结果，方便试错和玩。
- **进度与成就**：用浏览器本地存储记录「已完成」的章节，进度条和打勾会让她看到自己的进展，更容易坚持。

### 技术实现

- 纯前端：`index.html` + `styles.css` + `app.js`。
- 课程数据写在 `app.js` 的 `lessons` 数组里，后续要加课、改文案或改示例代码，只改这里即可。
- 运行 Python 依赖 Pyodide（通过 CDN 加载），首次打开可能需要几秒加载，之后即可在弹窗里运行代码。

## 如何使用

1. **本地打开**  
   直接双击 `index.html` 用浏览器打开即可。  
   若通过 `file://` 打开，部分浏览器可能限制脚本，若 Pyodide 加载失败，可改用下面方式。

2. **用本地服务器（推荐）**  
   在项目目录下执行：
   - Python 3：`python -m http.server 8080`
   - 或 Node：`npx serve .`  
   然后访问：`http://localhost:8080`（端口以实际为准）。

3. **给晗晗用**  
   把整个文件夹发给她，或部署到任意静态网站（如 GitHub Pages、Vercel 等），把链接发给她即可。

## 文件结构

```
晗晗的python的学习网站/
├── index.html    # 主页面
├── styles.css    # 样式
├── app.js        # 课程数据与交互（含 Pyodide 运行逻辑）
└── README.md     # 本说明
```

## 后续可以怎么做

- 在 `app.js` 的 `lessons` 里**新增课程**：复制一条 lesson 对象，改 `id`、`title`、`badge`、`encourage`、`content`、`code` 即可。
- **改情话/鼓励语**：直接改对应 lesson 的 `encourage` 或欢迎区的文案。
- **换配色/风格**：改 `styles.css` 里的 `:root` 变量和类名样式即可。
